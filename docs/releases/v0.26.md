# LLMGauge v0.26 Release Notes

LLMGauge v0.26 adds the foundation for tiered, claim-aware practical model evaluation.

## Highlights

- Added evaluation tier documentation.
- Added WumboLabs Practical Eval v1 prompt-quality guidance.
- Added the initial 10-prompt WumboLabs Practical Eval v1 seed suite.
- Added suite-aware manual score templates.
- Added score label validation against the selected rubric or suite vocabulary.
- Tightened the Practical Eval v1 seed prompts after local shakedown review.

## Practical Eval v1 status

`wumbolabs-practical-v1` is a seed Tier 2 suite. It is useful for prompt review, model shakedowns, scoring workflow validation, and early comparison design.

It is not yet a finished public benchmark and should not be used by itself to make broad model recommendations or rankings.

## Local shakedown

The v0.26 workflow was checked with a local Mellum2 Instruct run on the 10-prompt seed suite:

- 10/10 prompts completed.
- Result validation passed.
- Suite-aware score template generation worked.
- Score label validation worked.
- Scored report generation worked.
- Manual score average: 3.59 / 5.

The shakedown is evidence that the workflow functions end to end. It is not a public model recommendation.

## Claim boundary

v0.26 can claim:

- tiered evaluation documentation
- Practical Eval v1 seed-suite scaffolding
- suite-aware manual scoring
- rubric label validation
- successful local workflow shakedown

v0.26 cannot claim:

- automated judging
- finished benchmark status
- model recommendations
- public rankings
- production-grade statistical validity
